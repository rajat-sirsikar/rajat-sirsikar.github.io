# rajatsirsikar.com

Personal website of **Rajat Abhay Sirsikar** — Structural Engineer & PhD researcher (IIT Roorkee).

## Pages
- `index.html` — Home, timeline, about, affiliations
- `publications.html` — Publications (reverse chronological)
- `projects.html` — Selected structural & academic projects
- `karpathy.html` — Easter-egg page

## Stack
Pure static HTML + CSS (Inter font via Google Fonts). No build step required.

Deploy by pushing this folder to the `gh-pages` branch or the root of the GitHub Pages repository.
